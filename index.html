Pengaturan Firebase Database

Sesuaikan dengan data Firebase milik Anda sendiri agar data tersimpan selamanya.

Informasi Penting:

Secara default, aplikasi berjalan di mode demo sandbox. Jika Anda ingin menggunakan database Firebase Google milik Anda untuk dipasang di GitHub Pages, silakan masukkan kredensial config Firebase Anda di bawah ini.

Panduan Lengkap Firebase & GitHub Pages

Ikuti instruksi ini untuk me-online-kan aplikasi Anda.

1

Membuat Project Firebase Baru

Buka Firebase Console, buat project baru (bebas memberi nama, matikan analitik jika tidak perlu).

2

Setup Cloud Firestore Database

  • Klik menu Firestore Database di sidebar kiri Firebase Console Anda.
  • Klik tombol Create Database.
  • Pilih lokasi terdekat (misalnya asia-southeast2 untuk Jakarta).
  • Pilih Start in test mode agar aplikasi dapat membaca dan menulis data dengan mudah tanpa ribet aturan otentikasi ketat. Klik Create/Enable.
3

Menyalin Firebase Config SDK

  • Masuk ke Project Settings (ikon gerigi di samping 'Project Overview').
  • Di bawah tab 'General', gulir ke bawah ke bagian **Your Apps**.
  • Klik ikon **Web ()**, daftarkan aplikasi ini (nama bebas).
  • Salin objek JSON di dalam variabel firebaseConfig. Contoh format yang dibutuhkan: 
    {
  "apiKey": "AIzaSy...",
  "authDomain": "app-id.firebaseapp.com",
  "projectId": "app-id",
  "storageBucket": "app-id.appspot.com",
  "messagingSenderId": "...",
  "appId": "..."
}
  • Masukkan JSON tersebut ke dalam menu Set Database di aplikasi ini.
4

Menyimpan File & Upload ke GitHub Pages

  • Simpan file halaman web ini dengan nama index.html.
  • Buka akun GitHub Anda, buat sebuah repository baru (pilih Public).
  • Upload file index.html Anda ke repository tersebut.
  • Buka tab **Settings** di halaman repository tersebut, pilih sub-menu **Pages** di sebelah kiri.
  • Pada kolom "Build and deployment", pilih branch main atau master, lalu simpan.
  • Tunggu 1-2 menit, GitHub Pages akan menyajikan link live website aplikasi Anda!
